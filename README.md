# Config

```
“sticky”: (true/false) - Does the bar stick to the top of the page?
“mainText”: “This is the main text area”
“placement”: (“top”/”bottom”) - Bar to show at the top or the bottom of the screen.
“allowHide”: (true/false) - Users can hide the notification bar by click the X button.
“buttonUrl”: “​https://example.com”​-
"textColor": "#111111"
"buttonText": "Click this button!"
"openNewTab": "on" - Clicking button will open in new tab.
"wiggleButton": "on" - “Wiggle” button animation enabled (​https://daneden.github.io/animate.css/)​
   
 "backgroundColor": "#FFFFFF"
"brandingEnabled": "on" - Show ninjaconvert logo inside notification bar "buttonTextColor": "#333333"
"animateEntryExit": "on" - (​https://daneden.github.io/animate.css/)​ "buttonBackgroundColor": "#AAACCC"
```
